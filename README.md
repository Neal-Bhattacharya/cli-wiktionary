# cli-wiktionary
Wiktionary from the Terminal

```
Enter word (q to quit)
>Wiki
----------------------------------------
English
  Proper noun
       1. Abbreviation of Wikipedia.
German
  Noun
       1. wiki
Icelandic
  Proper noun
       1. wiki
Maori
  Proper noun
       1. a female given name, short for Wikitoria
----------------------------------------
Enter word (q to quit)
>badabing
Word not found.
```

## Dependencies
- python3
- requests
  
  `sudo pip3 install requests`
